## Overview
Project ini dibuat untuk mempermudah proses pengajuan judul hingga ujian skripsi secara digital. Sebagai **Frontend Engineer**, gue fokus pada performa dan pengalaman pengguna yang seamless.

## Tech Stack & Features
- **Frontend:** Next.js dengan App Router untuk SEO yang lebih baik.
- **Backend:** Laravel sebagai Robust API.
- **State Management:** React Context & SWR untuk sinkronisasi data real-time.

## Key Learnings
Selama ngerjain ini, gue belajar banyak tentang gimana cara handle file upload yang besar dan integrasi sistem autentikasi antar dua framework yang berbeda (Next.js & Laravel).
